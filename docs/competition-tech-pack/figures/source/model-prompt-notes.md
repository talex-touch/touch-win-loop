# 图片模型提示词索引

本目录保留的是图片模型生成用的主题说明与资料包映射，不把 HTML 视为最终图源。

## 生成原则

- `paper`：白底或极浅灰底，Nature / Science 子刊 schematic / graphical abstract 风格。
- `ppt`：深蓝黑背景，比赛路演用 16:9 技术架构图风格。
- 所有图统一强调：结构拆解、机制路径、输入输出关系、结果导向。
- 图面可见文字尽量使用少量大字中文，避免密集小字导致图片模型失真。
